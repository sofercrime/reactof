# useToggle

用于在两个状态值间切换的 React 状态钩子。

**核心特性**

* 设置默认状态
* 切换显示状态

## API

```javascript
const [state, toggle] = useToggle()

const [state, toggle] = useToggle(
  defaultValue: any = false,
)

const [state, toggle] = useToggle(
  defaultValue: any = false,
  reverseValue?: any,
)
```

### Result

| 参数     | 说明                                 | 类型                 |
|----------|--------------------------------------|----------------------|
| state  | 状态值                         | boolean              |
| toggle | 触发状态更改的函数,可以接受两个可选参数修改状态值 | (defaultValue: any = false, reverseValue?: any) => void |

### 参数

| 参数    | 说明                                         | 类型                   | 默认值 |
|---------|----------------------------------------------|------------------------|--------|
| defaultValue | 可选项，传入默认的状态值  | number \| string \| boolean \| undefined | -      |
| reverseValue | 可选项，传入取反的状态值  | number \| string \| boolean \| undefined | -      |
