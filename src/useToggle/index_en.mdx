# useToggle

React state hook that switch value in two states.

**Core Characteristics**

* Set default state.
* switch value in two states

## API

```javascript
const [state, toggle] = useToggle()

const [state, toggle] = useToggle(
  defaultValue: any = false,
)

const [state, toggle] = useToggle(
  defaultValue: any = false,
  reverseValue?: any,
)
```

### Result

| Property | Description                                         | Type                 |
|----------|--------------------------------------|----------------------|
| state  | state value                         | boolean              |
| toggle | Trigger state change, accept two optional parameters | (defaultValue: any = false, reverseValue?: any) => void |

### Params

| Property | Description                                 | Type                   | Default |
|---------|----------------------------------------------|------------------------|--------|
| defaultValue | Optional，set a default value  | number \| string \| boolean \| undefined | -      |
| reverseValue | Optional，set a reverse value  | number \| string \| boolean \| undefined | -      |
