## API

```
useEventListener(
  eventName:string;
  handler:Function;
  element:HTMLElement | Window
)
```

### Property

| Property    | Description     | type                   | default |
|---------|----------|------------------------|--------|
| eventName | event name | string | -      |
| handler | handle function | Function | -      |
| element | element on listening | HTMLElement or Window | window     |
