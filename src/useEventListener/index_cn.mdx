## API

```
useEventListener(
  eventName:string;
  handler:Function;
  element:HTMLElement | Window
)
```


### 参数

| 参数    | 说明     | 类型                   | 默认值 |
|---------|----------|------------------------|--------|
| eventName | 事件名称 | string | -      |
| handler | 处理函数 | Function | -      |
| element | 监听元素 | HTMLElement | Window | window     |
