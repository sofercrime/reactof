# useAPI

React side-effect hook that can run async functions.

## Usage

### Demo
```
import { useAsync } from '@umijs/hooks';

const Demo = () => {
  const {data, loading} = useAsync(asyncFunc);

  return loading ? ('loading...') : (<SomeComponent data={data}>)
};
```
