# useBoolean

跟踪布尔值的 React 状态钩子。

**核心特性**

* 设置默认状态
* 切换显示布尔值

## API

```javascript
const [state, toggle] = useToggle();

const [state, toggle] = useToggle(
  defaultValue?: boolean,
);
```

### Result

| 参数     | 说明                                 | 类型                 |
|----------|--------------------------------------|----------------------|
| state  | 状态值                         | boolean              |
| toggle | 触发状态更改的函数,可以接受一个可选参数修改状态值 | (value?: any) => void |

### 参数

| 参数    | 说明                                         | 类型                   | 默认值 |
|---------|----------------------------------------------|------------------------|--------|
| defaultValue | 可选项，传入默认的状态值  | boolean \| undefined | -      |
