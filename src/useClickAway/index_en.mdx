# useClickAway

A hook that elegantly manages click outside of target elements.

## API

```javascript
const ele = useClickAway(
  onClickAway: (event: KeyboardEvent) => void,
  ref?: RefType,
);
```

### Result

| Property | Description                                         | Type                 |
|----------|------------------------------------------|------------|
| ele     | when no param is passed, this ele will be listened      | -        |

### Params

| Property | Description                                 | Type                   | Default |
|---------|----------------------------------------------|------------------------|--------|
| onClickAway | Trigger Function  | (event) => void | -      |
| ref | Target Element  | () => HTMLElement | -      |
