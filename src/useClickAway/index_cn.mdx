## API

```javascript
const ele = useClickAway(
  onClickAway: (event: KeyboardEvent) => void,
  ref?: RefType,
);
```

### Result

| 参数     | 说明                                     | 类型       |
|----------|------------------------------------------|------------|
| ele     | 当未传入任何参数时，将 ele 绑定给需监听的节点      | -        |

### Params

| 参数    | 说明                                         | 类型                   | 默认值 |
|---------|----------------------------------------------|------------------------|--------|
| onClickAway | 触发事件的函数  | (event) => void | -      |
| ref | 目标元素  | () => HTMLElement | -      |
