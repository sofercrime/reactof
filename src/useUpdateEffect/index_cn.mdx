# useUpdateEffect

useUpdateEffect 与 useEffect API 一致，不过它会忽略第一调用（mount 时），也就是 useUpdateEffect 只会在更新时执行。

**核心特性**

* 适用于 hooks 业务场景下的应用
* effect 可选的清理机制,清除副作用

## API

```javascript
useUpdateEffect(
  effect: () => void,
  deps?: deps,
)
```

### 参数

| 参数    | 说明                                         | 类型                   | 默认值 |
|---------|----------------------------------------------|------------------------|--------|
| effect | 可执行函数  | function | -      |
| deps | 可选项，传入依赖变化的对象  | array \| undefined | -      |
