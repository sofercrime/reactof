# useUpdateEffect

一个作用于组件 didMount 阶段以后的 useEffect hook。

**核心特性**

* 适用于 hooks 业务场景下的应用
* effect 可选的清理机制,清除副作用

## API

```javascript
useUpdateEffect(
  effect: () => void,
  deps?: deps,
)
```

### 参数

| 参数    | 说明                                         | 类型                   | 默认值 |
|---------|----------------------------------------------|------------------------|--------|
| effect | 可执行函数  | Function | -      |
| deps | 可选项，传入依赖变化的对象  | array | array \| undefined | -      |
