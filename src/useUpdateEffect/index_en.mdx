# useUpdateEffect

A hook only works after component mounted.

**Core Characteristics**

* Suitable for use in hooks business scenarios
* Optional effect hook cleanup mechanism to remove side effects

## Examples

<JackBox jsCode={Demo1CodeJsx} tsCode={Demo1CodeTsx} demoName='basic usage' description='It is exactly the same in use as useEffect, except that the life cycle is placed after the component didMount'>
  <Demo1 />
</JackBox>

## API

```javascript
useUpdateEffect(
  effect: () => void,
  deps?: deps,
)
```

### Params

| Property | Description                                                        | Type                   | Default |
|---------|----------------------------------------------|------------------------|--------|
| effect | Executable function  | Function | -      |
| deps | Optionally, pass in objects that depend on changes  | array | array \| undefined | -      |
