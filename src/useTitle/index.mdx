# useTitle

React side-effect hook that set title of the page.

## Usage

### Demo
```
import {useTitle} from '@umi/hooks';

const Demo = () => {
  useTitle('Hello world!');

  return null;
};
```

## Reference

```
useTitle(title: string);
```
